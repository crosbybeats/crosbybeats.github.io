How to make these epic progress bars:



  • 1. Start by setting up your HTML structure with a !DOCTYPE html, html, head, and body tags
  • 2. Inside the head tag, make sure to add a style tag to add your CSS
  • 3. Setup margins, paddings, and set box-sizing: border-box for all elements
  • 4. Style the body to use flexbox for centering its content vertically and horizontally
  • 5. Create a class, in this case 'progress-bar-container', to contain each progress bar and its label
  • 6. Style the h2 elements within the progress-bar-container to define the font, color, spacing, or anything else that you want
  • 7. Inside each progress-bar, use a span element with a class percentage to represent the filled portion of the bar
  • 8. Style the 'percentage' class to set the background color/colors, border, and animation for the progress effect
  • 9. Use the @keyframes tag to animate the width of the percentage elements from 0 to their defined widths over a specific amount of time
